# Ümit Diet Companion

A personal AI-powered health dashboard developed with SwiftUI for iPhone.

The goal of this project is to replace multiple health applications with a single, intelligent dashboard that combines daily health metrics, nutrition tracking and AI coaching.

---

## Vision

Instead of switching between:

- Garmin Connect
- Apple Health
- Notes
- ChatGPT
- Calorie tracking apps

the user manages everything from one application.
